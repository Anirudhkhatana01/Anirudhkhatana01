## 🔥 Featured Projects

### 📈 TechnoEdge Financial Analysis Dashboard

Developed an interactive **Power BI** dashboard to analyze financial performance using Excel datasets. Performed data cleaning, transformation, and modeling with **Power Query**, and built **DAX** measures for Sales, Profit, Gross Sales, Discounts, Profit Margin, and KPI reporting. The final dashboard uses cards, slicers, drill-through reports, charts, and tables to surface business insights across products, customer segments, countries, and yearly performance.

**Tech used:** **Power BI** • **Power Query** • **DAX** • **Excel**
